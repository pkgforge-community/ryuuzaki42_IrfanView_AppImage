!*! Please use always the current IrfanView and PlugIn version !*!


- How to install IrfanView PlugIns?

  1) Visit IrfanView's homepage: http://www.irfanview.com
  2) Go to "Download->PlugIns"
  3) Download PlugIns
  4) Click on the PlugIn file
  5) PlugIns will be installed into IrfanView "PlugIns" directory


Download:

You can download all PlugIns from the Internet:
   http://www.irfanview.com/plugins.htm
   http://www.irfanview.net/plugins.htm

Special PlugIns:

* 1. IrfanView Shell Extension - This PlugIn shows a Context menu for some IrfanView operations in Windows Explorer.

Normal PlugIns:

* 1. ALTALUX.DLL - This PlugIn allows IrfanView to use AltaLux image effect.
                   Thanks to Stefano Tommesani!

* 2. AVIF.DLL - This PlugIn allows IrfanView to read AVIF files.

* 3. AWD.DLL - This PlugIn allows IrfanView to read Artweaver files.
               Thanks to Boris Eyrich.

* 4. BABACAD4IMAGE.DLL - This PlugIn allows IrfanView to read DXF and DWG CAD files.
                         Thanks to BabaCAD Software guys: http://www.babacad.com

* 5. B3D.DLL - This PlugIn allows IrfanView to read BodyPaint 3D files.

* 6. BURNING.DLL - This PlugIn allows IrfanView to burn slideshow to a CD/DVD/BD.
                   Windows XP SP3 or later required.

* 7. BURNINGOLD.DLL - This PlugIn allows IrfanView (32-bit) to burn slideshow to a CD.
                      Windows XP or later required, Nero option can burn Data DVD and VCD.

* 8. CAMRAW.DLL - This PlugIn allows IrfanView to read Digital Camera RAW files (thanks to Alex Tutubalin).
                  (ARW CR3 CR2 CRW DCR DNG ERF KDC MRW NEF NRW ORF PEF RAF RAW RW2 RWL SRF SRW X3F MOS)

* 9. CRW.DLL - This (32 bit) PlugIn allows IrfanView to read Canon CRW/CR2 files using Canon DLLs.
               Note: this PlugIn requires additional Canon DLLs: http://irfanview.info/plugins/canon_crw.zip
               (check "Readme_Canon.txt" for instructions and install!)

* 10. CADIMAGE.DLL - This PlugIn allows IrfanView to read CAD formats: DXF, DWG, HPGL, CGM, SVG.
                    Check "http://www.cadsofttools.com/download/irfanviewplugins.zip" (32-bit) for newest versions and licenses.
                    Check "http://www.cadsofttools.com/download/irfanviewplugins_x64.zip" for 64-bit PlugIns.
                    (Shareware, Third party PlugIn)

* 11. DICOM.DLL - This PlugIn allows IrfanView to read Dicom medical files. 
                 Thanks to Wolfgang Krug and Chris Rorden.
                 Note: DICOM/ACR/IMA format hasn't a specific header and IrfanView can't recognize all such images. 
                 If you want to read Dicom/ACR/IMA images with IrfanView, the file extension MUST BE ".dcm", ".acr" or ".ima" !

* 12. DJVU.DLL - This PlugIn allows IrfanView to read DjVu files.

* 13. DPX.DLL - This PlugIn allows IrfanView to read DPX/CIN (Digital Picture Exchange/Cineon) files.

* 14. ECW.DLL - This (32 bit) PlugIn allows IrfanView to read/write ECW files (Enhanced Compressed Wavelet).

* 15. EFFECTS.DLL - This PlugIn contains some image effects.
                    Thanks to GENiEBEN for many effects!

* 16. EMAIL.DLL - This PlugIn allows IrfanView to send images as emails.

* 17. EXR.DLL - This PlugIn allows IrfanView to read EXR files.

* 18. FACEDETECT.DLL - This PlugIn allows IrfanView to use the Face Detection feature from Thumbnails window.

* 19. FFACTORY.DLL - This PlugIn allows IrfanView to use Filter Factory 8BF files (Photoshop PlugIns).
                     The DLL is based on the Filter Factory library from Michael Johannhanwahr: www.icnet.de.
                     Thanks Michael!

* 20. FILMSIM.DLL - This PlugIn allows IrfanView to change the picture colors using a CLUT file (Color-Lookup-Table). 
                    For this feature you must first download and extract the Film Simulation file 
                    (irfanview_film_sim_plugin.zip) from the PlugIns website.

* 21. FLASH.DLL - This PlugIn allows IrfanView to play Macromedia's Flash/Shockwave/FLV files.
                  It uses the Flash Player component installed in Internet Explorer.
                  Thanks to Remio!

* 22. FLIF.DLL - This PlugIn allows IrfanView to read FLIF (Free Lossless Image Format) files.

* 23. FORMATS.DLL - This PlugIn allows IrfanView to read additional image formats, like: 
                  - PCX/DCX - Zsoft Paintbrush
                  - PSP - Paint Shop Pro format
                  - IFF/LBM format (Amiga Interchange File Format)
                  - G3 Fax, CLP, RAS/SUN, SGI, GEM-IMG, Mosaic, YUV, XBM, XPM, GLCD, MPO format
                  - BLP format (BLP-2)
                  - MAG format (MAKI)
                  - WBMP - WAP Bitmap
                  - Bio-RAD Confocal image format
                  - RLE format (Utah RLE Image)
                  - TTF format (True Type Font)
                  - RAW format (RAW data)
                  - DDS format (Direct Draw Surface)
                  - SFW format (Seattle Film Works JPG file)
                  - CAM format (Casio CAM JPG file)
                  - KDC format (Kodak Digital Camera files)
                  - WAD format (WAD3 game files)
                  - WAL format (Quake 2 textures)
                  - FITS format (Flexible Image Transport System)
                  - PVR format (DreamCast textures)
                  - support for old formats like: Amiga, Atari, C64, ZX Spectrum etc. (based on file extension)

                  Note: G3/Mosaic/XBM/XPM/WBMP format hasn't a specific header and IrfanView 
                  can't recognize all such images. If you want to read such with IrfanView, 
                  the file extension MUST BE ".g3"/".mos"/".xbm"/".xpm"/".wbmp" !

* 24. FPX.DLL - This PlugIn allows IrfanView (32 bit) to read FlashPix format.

* 25. FTP.DLL -  This PlugIn allows IrfanView to transfer files from Thumbnails window using FTP.

* 26. HDP.DLL - This PlugIn allows IrfanView to read Jpeg-XR/HDP/WDP files (Microsoft HD Photo).

* 27. ICONS.DLL - This PlugIn contains special icons for IrfanView file associations.
